## 第 1 幕 · 一句没说出口的话（场景痛点）

**封面副标题**：「想说，却说不出口」——这是中国家庭最普遍的情感困境。

**核心数据**（PPT 引用，需在备注页注明来源）：
- 中国 65% 的成年子女与父母"几乎只聊吃饭和身体"（《2023 家庭沟通白皮书》参考）
- 60+ 岁父母平均每天使用手机 3.4 小时，但与子女的"主动沟通"不到 6 分钟
- 微信"对家人说不出口"的撤回率，是同事会话的 3 倍

**用户原话（用户研究片段，PPT 引用即可）**：
- "妈，你少操点心" —— 删除，重打 —— "嗯，我挺好的"
- "爸生日想给他唱首歌" —— 没发出去
- "对不起" —— 没勇气说

**痛点收口**：东方家庭含蓄、关心常被"翻译"成催促，导致代际之间的情感损耗。

## 第 2 幕 · 安语是什么（产品定义）

**一句话定义**：  
> 安语 = 一个让"说不出口的话"被东方意象接住、被 AI 温柔润色、被 VIVO 原子组件准时送达家人手机桌面的家庭情感传递 APP。

**核心模型**：「意象 → 诗意 → 投递 → 桌面接住」

四步用户旅程：
1. 选一个意象（一盏暖灯 / 一杯热茶 / 一封家书 / 雨后春笋 …）
2. AI 把这份心意润色为东方诗意短句
3. 投出 → 自动送达家人手机的「VIVO 原子组件」
4. 家人在桌面"接住"这份心意，无需打开 APP

**双向角色**：
- 孩子端：含蓄表达 + AI 润色，把粗糙情绪转成温柔表达
- 父母端：默认更大字号、更慢节奏，"接住"被翻译过的关心

## 第 3 幕 · 三个 VIVO 原子组件方案（比赛主体）

> 这一幕需要现场演示 manifest.xml + Web 原型 + 桌面动效。

**组件一 · 心湖**（2x2）
- 接收人：父母 / 子女
- 内容：家人当下的情绪天气（晴 / 阴 / 雨 / 雪）
- 行为：双击可以发出一份"我想你了"无字心意
- 价值：让"今天 TA 心情怎么样"成为家庭日常感知

**组件二 · 心意**（4x2）
- 接收人：父母
- 内容：孩子寄出的意象 + AI 润色短句（一盏暖灯 / 茶香）
- 行为：长按"接住" → 桌面播放一段 1.5 秒的水墨涟漪
- 价值：父母不打开任何 APP，就能收到孩子今天的心意

**组件三 · 写一句**（2x2）
- 接收人：子女
- 内容：父母端的"一键说一句话"，无需输入完整句子
- 行为：点击预设意象 → 自动生成一句话 → 投递到孩子组件
- 价值：解决父母"不会写、不想打字"的真实门槛

**组合联动**（比赛差异化关键）：
- "我寄出一杯热茶" → 父母端「心意」组件亮起 → 父母「写一句」回一句"路上小心" → 孩子端「心湖」涟漪变亮
- 全链路无需打开 APP，全部在桌面完成

## 第 4 幕 · 我们和别人不一样（差异化）

| 维度 | 微信家庭群 | 念念 / 暖心 类亲情 APP | **安语** |
| --- | --- | --- | --- |
| 表达门槛 | 高（要说全句） | 中（要写、要选） | **低**（一个意象就行） |
| 美学语言 | 通用 emoji | 通用插画 | **东方意象 + 几何图标** |
| 父母触达 | 要打开微信 | 要打开 APP | **桌面原子组件接住** |
| AI 角色 | 无 | 翻译 / 总结 | **诗意润色 + 双向翻译** |
| 商业空间 | 已饱和 | 较窄 | **VIVO 系统级 + 银发 ToB** |

**差异化锚点（PPT 重点标红）**：
- 唯一一个把"东方含蓄表达"明确做成设计语言的产品
- 唯一一个把"原子组件"作为主要交付载体而非附属功能的产品
- 唯一一个为"双向、非对等家庭关系"专门设计两套视觉的产品

## 第 5 幕 · 商业可行性（落地路径）

**短期（比赛阶段）**：
- 与 VIVO OriginOS 原子组件生态深度绑定，作为情感类官方推荐
- 通过 VIVO 应用商店首发，覆盖银发用户的"开机即用"场景

**中期（半年内）**：
- 与"老年大学 / 社区养老中心"合作，把父母端做成轻量入口
- 与情绪心理类机构合作"贴心翻译"专业版

**长期（一年以上）**：
- 跨端：HarmonyOS / iOS Widget / MIUI 同步铺开
- ToB：保险公司、银发健康服务方的家庭情感连接基础设施

**最小可验证指标（MVP 验证）**：
- 单日意象寄出数 / 用户
- 父母端组件"接住率"
- 7 日双向回信率（衡量"情感闭环"）
